Free/Libre Open Source Software Design

People who roam the web and creates and designs different parts of it tend to have a good eye towards both Free/Libre Open Source Software and Creative Commons. This is also true for hackers, teachers, librarians and many others.

But as soon as you say "design", someone tries to force you to use any product from Adobe. It might be tools that many use, but it is not the tools you have to use.

So where can we find the open and free tools to do our craft? Where are the people who use these tools to teach and spread the knowledge of the web?

Perchance this might be a place to find info about this very important part of the world wide web.

If you belive that software should be open and free, than this is an important issue for you. So either if it is by personal choice or organisation policy, there are tools for you!
